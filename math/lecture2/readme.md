- ## 最大公约数(GCD GCF)和最小公倍数(LCM)(Greatest common divisors and least common multiples)

两个数的最大公约数与最小公倍数的求解方法：
()将两个数分别各自分解质因数：
()每一个质数，取较小的指数，相乘得到最大公约数：
每一个质数，取较大的指数，相乘得到最小公倍数
